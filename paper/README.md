# Accompanying Paper

The technical paper for this project provides a detailed analysis including:

- Architectural design decisions and their rationale
- Comparison with PFMVG (He et al., 2025) framework
- Detailed training analysis and ablation findings
- Discussion of mode collapse patterns and limitations

**Title:** *Medical Image Captioning with ViT-Base and Phi-2 using LoRA Fine-Tuning*  
**Author:** Rana Rishith Musunuri  
**Affiliation:** Department of Information Technology, Manipal University Jaipur, India
